# Interactive Line Chart

An interactive Line Chart to visualize A/B test statistics.

## Tech Stack

React + TypeScript
Chart library - Recharts
CSS Modules for component styling.

## Local Setup

1.  **Install dependencies:**
    ```bash
    pnpm install
    ```

2.  **Run development server:**
    ```bash
    pnpm dev
    ```

3.  **Build for production:**
    ```bash
    pnpm build
    ```

## Deployment

Deploy to GitHub Pages.

1.  Update `vite.config.ts` with your base URL if needed (e.g., `base: '/repo-name/'`).
2.  Run `pnpm build`.
3.  Deploy the `dist` folder.
